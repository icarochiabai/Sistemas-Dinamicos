Visualização do Caos

Nessa animação utilizamos um número real para determinar uma curva.
Exemplo:
Vamos desenhar pontos sobre uma reta horizontal espaçados em 1 unidade. Utilizando o número 1.30192 como exemplo, vamos iterar sobre seus digitos para determinar as rotações que serão feitas enquanto desenhamos a reta.
Para o dígito 1, iremos realizar uma rotação de 1 radiano e desenharemos um ponto na reta.
Para o dígito 3, iremos realizar uma rotação de 3 radianos e desenharemos outro ponto na reta.
Repetimos esse processo até chegarmos ao final do número. Um detalhe importante é que se o dígito for par, sua rotação será realizada no sentido horário, caso contrário, é realizada no sentido anti-horário.

Isso faz com que números próximos, como 1.301927419548 e 1.301856932164, apresentem curvas bastante distintas.

A animação mostra as diferentes curvas geradas pelos números √6, √7, √8, com 2000 dígitos de precisão, sendo:
Curva vermelha: √6
Curva verde: √7
Curva azul: √8

Deixando esse sistema evoluir com o tempo, obtemos a animação ao lado.

Arraste a animação para esquerda/direita para alterar o coeficiente de rotação.
Gire a roda do mouse para alterar a escala.
Aperte o botão direito do mouse para recomeçar a animação.
